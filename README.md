# 💻 Research Simplified — Client

Research Simplified (client) is a **modern React-based frontend** designed to make academic papers easier to consume and interact with.

It enables users to **upload research papers, receive section-wise simplified outputs**, and interact through a **context-aware AI assistant** — all powered by a **real-time streaming experience**.

---

## ✨ Key Features

### ⚡ Real-time Streaming Experience
- Simplified content starts rendering within **8–10 seconds of upload**
- Token-by-token streaming for:
  - Section-wise simplification
  - Chat responses
- Provides fast, continuous feedback instead of waiting for full results

---

### 📄 Intelligent Paper Simplification
- Processes uploaded research papers into logical sections
- Focuses on extracting and simplifying **meaningful content**
- Designed to improve readability of dense academic material

---

### 💬 Context-Aware Chat Assistant
- Ask questions about:
  - Specific sections
  - Entire paper
- Maintains conversational context
- Acts as an interactive layer on top of the processed document

---

### 📑 Built-in PDF Viewer
- Integrated PDF viewer for reading original documents
- Seamless switch between:
  - Original paper
  - Simplified output
- Keeps users grounded in source material

---

### 🔐 Authentication
- Google OAuth login
- Secure session handling
- Personalized experience for users

---

### 🗂️ State Management
- Managed using **Redux Toolkit**
- Efficient data fetching with **RTK Query**
- Structured and scalable client-side architecture

---

### 🔄 Robust Error Handling
- Clear and user-friendly error states
- Retry mechanisms for failed operations
- Non-blocking UI interactions

---

### 📱 Responsive UI
- Fully responsive across devices
- Clean and minimal interface
- Optimized for readability and usability

---
